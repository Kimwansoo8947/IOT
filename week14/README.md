###
Wi-Fi 구성
AP (Access Point)와 station(STA)으로 구성
  - AP: 유무선 공유기
  - STA: 노트북, 스마트 폰 등

Wi-Fi Service Set
 - BSS ex) 가정
    -  AP가 없으면 ad hoc 모드
    -  AP가 있으면 infrastructure 모드 => 일반적으로 사용

- ESS
  - 두 개 이상의 BSS들이 모여서 구성 => BSS를 여러개 묶어서


MAC 방식
  -PCF => 중앙 집중식
  -DCF => CSMA/CA 사용 (Wi-Fi MAC 방식 중에 주로 사용됨)
